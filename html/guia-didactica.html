GUÍA DIDÁCTICA

Descripción

Este proyecto está diseñado para el alumnado de 3º de Educación Secundaria Obligatoria en la materia de Educación Física, con el objetivo de desarrollar tanto competencias motrices como digitales a través del análisis del movimiento y la mejora de la ejecución.

 
RELACIÓN ENTRE TAREAS Y CRITERIOS DE EVALUACIÓN


A continuación, se muestra la relación entre las actividades del proyecto y los criterios de evaluación trabajados:

Actividad 1: Observación y análisis inicial
👉 Criterio de evaluación: Identificar aspectos técnicos correctos en la ejecución de ejercicios físicos.

Actividad 2: Grabación del ejercicio
👉 Criterio de evaluación: Participar activamente en actividades físicas utilizando de forma adecuada los recursos disponibles.

Actividad 3: Análisis del movimiento
👉 Criterio de evaluación: Analizar la propia ejecución y detectar errores.

Actividad 4: Propuesta de mejora
👉 Criterio de evaluación: Aplicar estrategias para mejorar la ejecución motriz.

Actividad 5: Creación del producto final
👉 Criterio de evaluación: Elaborar un producto digital utilizando herramientas tecnológicas.

Actividad 6: Exposición y reflexión
👉 Criterio de evaluación: Comunicar de forma clara el proceso de aprendizaje y reflexionar sobre la propia evolución.
 
Metodología
Se emplea una metodología activa basada en el aprendizaje por proyectos, donde el alumnado es protagonista de su propio aprendizaje.

Se fomenta:

- El trabajo cooperativo
- La reflexión sobre la práctica
- El uso de herramientas digitales
- La autonomía del alumnado
- El docente actúa como guía, acompañando el proceso y ofreciendo feedback continuo.

 
Atención a la diversidad (DUA)
El proyecto se adapta a las necesidades del alumnado mediante:

- Diferentes formas de acceso a la información (vídeos, explicaciones, ejemplos)
- Posibilidad de elegir herramientas digitales (Canva, Genially…)
- Trabajo en grupo para facilitar el apoyo entre iguales
- Adaptación del ritmo de trabajo
 
Uso de las tecnologías digitales
Las tecnologías se utilizan como herramienta para:

- Analizar el movimiento
- Crear contenido digital
- Mejorar el aprendizaje

Se promueve un uso:

- Responsable
- Seguro
- Crítico
 
Observaciones para el profesorado
Es recomendable supervisar el uso de dispositivos durante las grabaciones
Se debe garantizar el respeto a la privacidad del alumnado
Se puede adaptar el nivel de exigencia según el grupo
El proyecto permite ajustes en función del tiempo disponible
